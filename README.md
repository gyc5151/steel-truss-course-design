# 钢结构屋架课程设计助手

在线使用：<https://gyc5151.github.io/steel-truss-course-design/>

这是一个无需构建的静态网页工具，用于《钢结构设计》梯形钢屋架课程设计的方案查表、荷载计算、屋架计算简图、杆件内力、截面初选、节点设计辅助和计算书草稿导出。

## 使用方式

直接用浏览器打开 `index.html`，或在本目录运行：

```bash
python3 -m http.server 5174
```

然后访问：

```text
http://localhost:5174/
```

## 主要功能

- 输入方案号自动查表：跨度、钢材、焊条、雪荷载、保温层、积灰荷载。
- 可手动覆盖跨度、荷载、材料强度和分项系数。
- 自动计算永久荷载、可变荷载、三类荷载组合和上弦节点荷载。
- 按内置梯形屋架模型进行平面桁架刚度法分析。
- 输出杆件内力表、截面初选表、节点焊缝长度和节点板厚度建议。
- 屋架计算简图支持点击杆件和节点查看编号、内力、长度、控制组合与相连杆件。
- 支持导出荷载 CSV、杆件 CSV、屋架 SVG、计算书 Markdown 和本机 JSON 数据。
- 支持浏览器打印或另存为 PDF。

## 注意

本工具用于课程设计草稿和画图参考。最终提交前，应对照老师给定的屋架几何尺寸图、内力系数图、教材型钢表和 GB 50017-2017 复核。
